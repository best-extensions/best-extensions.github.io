CHROME EXTENSION METHOD - Manual Installation Guide

1. Download the ZIP

Download the extension ZIP file from THIS LINK, extract it on your computer, and keep the unpacked folder ready.

2. Open Chrome Extensions

Open Chrome and go to chrome://extensions/.

3. Enable Developer Mode

Turn on Developer mode in the top-right corner of the page.

4. Load Unpacked

Click Load unpacked, select the extracted folder, and the extension will be installed immediately.

Tip: If you update the extension files later, simply click Reload from the Chrome extensions page.
